## Local 

1. Clone the repository:
    ```bash
    git clone https://github.com/sahalhes/bookrecom.git
    ```

2. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the `app.py` file using streamlit:
    ```bash
    streamlit run app.py
    ```
